# Paradis Hotel

Paradis Hotel is a simple Java + Maven hotel management system skeleton.

## Scene Builder Prototype

This project UI runs using programmatic JavaFX code from controller classes.

A separate Scene Builder prototype file is included for demonstration/documentation:
- `src/main/resources/fxml/DashboardPrototype.fxml`

This FXML file is not wired into runtime flow, so current application behavior remains unchanged.

## Project Structure

- `src/main/java/com/paradis/hotel/main` - Application entry point
- `src/main/java/com/paradis/hotel/model` - Domain models
- `src/main/java/com/paradis/hotel/dao` - Data access classes
- `src/main/java/com/paradis/hotel/controller` - Controller layer
- `src/main/java/com/paradis/hotel/util` - Shared utilities
